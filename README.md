# TCS-WWKC-KF

This repository contains the **Knowledge File** for the [*Who Was Karmein Chan?*](https://thecruelsong.com/) project: a structured, source-driven research dataset dedicated to documenting the life, disappearance, murder, and memory of Karmein Chan. Its purpose is to preserve verified information, separate fact from inference, track sources transparently, and provide both a canonical JSON master file and token-efficient exports for responsible use with large language models.
